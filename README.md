# 📖 SelectCode-CSS-Challenge <a name="about-project"></a>

A simple HTML and CSS project replicating a Figma design as part of the SelectCode challenges. For this project I decided to go with a mobile first development approach which allowed me to build on the responsiveness from smaller screens to larger screens. i found it a little challenging to make the sidebar respond to being collapsed and expanded but after tweaking with a little javascript, I managed to make it fully responsive

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 git
 npm
 node
 Your favorite text editor
```

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/Shisui6/selectcode-CSS.git
```

### Install

Install this project with:

```sh
  cd selectcode-CSS
  npm install
```
